## MODIFIED Requirements

### Requirement: Rewrite Report Covers Key Supporting Elements

When supporting elements materially affect the condensed manuscript, the rewrite report MUST reflect them.

#### Scenario: The rewrite report is reviewed after a supporting-elements-heavy draft
- **WHEN** key figures, key tables, key citations, or bibliography decisions shaped the final draft
- **THEN** `rewrite-report.md` includes notes for those key elements
- **AND** the report makes clear whether they were preserved, rewritten, merged, omitted, or left as placeholders
